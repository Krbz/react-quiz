# react-quiz - Work in progress
Quiz based on [React](https://github.com/facebook/react)

### 1. Install dependencies
```sh
$ npm install 
```
or
```sh
$ yarn
```

### 2. Run server
```sh
$ npm start
```

### 3. Open borwser
```
localhost:3000
```
